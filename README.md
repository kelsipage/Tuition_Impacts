# Tuition_Impacts
Exploring the impacts that cyber and charter school tuition has on public school districts in Pennsylvania.

## Summary of Insights

There is a lot of discussion on the impacts that charter school per student tuition rates have on public school districts. This rate has been shown to create inequitable funding, benefiting cyber charter schools in particular. Organizations and impacted school districts have been working together to make adjustments to this formula to create a more equitable tuition rate that would benefit all students in Pennsylvania. These data explore the specific impacts that cyber charter school tution rates have had on public school districts in Pennsylvania.
